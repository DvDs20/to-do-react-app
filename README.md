# to-do-react-app
1. Prieš paleidžiant parašome komandą "npm install", kuri suinstaliuoja visus reikalingus package'us.

3. Tik svarbiausia, jog paleistumėte iš pradžių back-endą (https://github.com/DvDs20/to-do-app-spring-boot)
